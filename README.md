# Decidim::Questions

Questions / Views / Inputs based on decidim-proposals.

## Usage

Questions will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'decidim-questions
```

And then execute:

```bash
bundle
```
